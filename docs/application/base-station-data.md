# eduroam JP 機関情報・基地局データについて

出典: [NII学術認証推進室 Wiki](https://nii-auth.atlassian.net/wiki/spaces/NIIninsho/pages/44270514)
取得日: 2026-03-16

## 概要

eduroam JP申請システムに基地局データ作成支援機能が実装されている。従来のXML形式ファイルではなく、スプレッドシートで直感的に作成できる。機関情報データとのリンクにより入力効率が向上している。

## 基地局データの形式

基地局データはeduroam.orgの定める「eduroam database」形式に準拠する。現在、厳密な運用ルールは定められていない。

## キャンパス単位での記述

キャンパス単位でeduroam利用可能場所を記述する場合の必須項目：

| 項目 | 値例 | 記入ポイント |
|------|------|-------------|
| 基地局名称（英語） | X Campus, A University | キャンパス名 |
| 基地局名称（日本語） | A大学 Xキャンパス | 日本語表記 |
| 基地局緯度・経度 | 35.692712,139.757938 | 代表的場所の座標 |
| 基地局運用水準 | 本格運用 | 機関方針に応じて選択 |
| 基地局種別 | エリア | キャンパス表示方式 |
| 基地局所在地（英語） | Chiyoda-ku, Tokyo | 都道府県市区町村 |
| 基地局所在地 町名番地（英語） | 5-963 Hitotsubashi | 詳細住所 |
| SSID | eduroam | ネットワーク識別子 |

## よくある質問

**Q: 「機関Web URL」と「機関ポリシURL」とは？**

A: eduroam databaseでは、機関Web URLは「サービス関連情報ページ」、機関ポリシURLは「運用ポリシ掲載ページ」。対応ページがない場合は機関URLを記入し、後日更新する。
