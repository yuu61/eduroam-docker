# 各サービスの学認参加要否等

出典: [NII学術認証推進室 Wiki](https://nii-auth.atlassian.net/wiki/spaces/NIIninsho/pages/44270704)
取得日: 2026-03-16

## 概要

NII学術認証推進室が提供する各サービスにおける、学認への参加の要否をまとめたもの。

## eduroam JP 関連サービス

| サービス | 学認参加要否 | 必要なIdP | 備考 |
|----------|-------------|-----------|------|
| **eduroam JP 申請システム** | **不要** | 学認参加IdPまたはOrthros | Orthrosでログイン可能 |
| **eduroam JP 認証連携IDサービス** | **必要** | 学認参加IdP | 学認参加が前提 |

## 認証関連サービス

| サービス | 学認参加要否 | 必要なIdP | 備考 |
|----------|-------------|-----------|------|
| 学認（運用フェデレーション） | 不要（未参加はOrthros利用） | 学認参加IdPまたはOrthros | - |
| テストフェデレーション | 不要（未参加はOrthros利用） | 学認参加IdPまたはOrthros | - |
| 学認申請システム | 不要 | 学認参加IdPまたはOrthros | - |
| mAP Core | 必要 | 学認参加IdP | - |
| NII FileSender | 必要 | 学認参加IdP | - |
| meatmail | 必要 | 学認参加IdP | Orthrosでも一部利用可 |

|----------|-------------|-----------|------|

| サービス | 学認参加要否 | 必要なIdP | 備考 |
|----------|-------------|-----------|------|
| GakuNin RDM | 原則として必要 | 学認参加IdPまたはOrthros | - |
| 学認LMS | 原則として必要 | 学認参加IdPまたはOrthros | - |

## 注釈

- Orthrosはサービスレベルが低いため、学認参加IdP利用が推奨される
- サービス利用時に学認参加IdPを利用する場合は別途学認参加が必要

## 本プロジェクトへの適用メモ

- **eduroam JP申請システム**: 学認不要 → **Orthrosでログイン可能**（本プロジェクトはこのルート）
- **認証連携IDサービス**: 学認必要 → 本プロジェクトは自前RADIUS方針のため利用しない
- 将来的にUPKI証明書を利用する場合も、学認参加は不要（UPKI証明書はeduroamとは別の申請経路）
