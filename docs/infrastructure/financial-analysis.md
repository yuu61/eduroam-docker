# 学校法人コンピュータ総合学園 財務状況レポート

> 出典: 2024年度（令和6年4月1日〜令和7年3月31日）事業報告書・財務諸表・監査報告書

---

## 1. 総括

学校法人コンピュータ総合学園の2024年度決算は、**総資産414.4億円**、**純資産395.5億円**、**当年度収支差額+12.6億円**と、極めて健全な財務状態を維持している。負債比率は4.8%と極めて低く、流動比率は680%超と資金流動性も十分に確保されている。借入金はゼロであり、無借金経営を継続中。

ただし、学生生徒等納付金は3年連続で減少傾向にあり、収入構造の変化に留意が必要。

### 5-4. 学生納付金の減少傾向

```
2020: ████████████████████████  24.3億円
2021: ██████████████████████████  26.8億円
2022: ███████████████████████████  27.8億円 ← ピーク
2023: █████████████████████████  26.0億円 (△6.5%)
2024: ███████████████████████  23.5億円 (△9.6%)
```

2022年度ピーク（27.8億円）から2024年度（23.5億円）まで **△15.5%の減少**。

学校関係者評価報告書（2025年10月）は主因を明示:
> 「在学生数については**18歳人口の減少を受け日本人入学生が減少傾向**にあり、外国人留学生の増加もあるが、総定員に対して若干の減少となった。」

- 神戸電子専門学校の在籍者1,889名（2024/5/1）が法人の学生数の大部分を占めるため、専門学校側の日本人入学者減が納付金減少に直結
- 外国人留学生は増加傾向（非漢字圏からの入学が顕著）だが、日本人学生の減少を補うには至っていない
- 学生生徒等納付金比率も74.5%（2021）→ 67.6%（2024）と低下し、収入源としての依存度が下落

## 7. 財務上の特徴と強み

### 7-1. 極めて強固な財務基盤
- **無借金経営**: 借入金残高ゼロ。利息支出もゼロ
- **潤沢な手元資金**: 現金預金 **114.3億円** + 有価証券 **142.3億円** = 流動性資産合計 **約256.6億円**
- **低い負債比率**: 総負債19.0億円に対し純資産395.5億円（負債比率4.8%）

### 7-2. 安定した資産運用収入
- 有価証券（142.3億円）からの受取利息・配当金が **年間4.0億円**（運用利回り約2.8%）
- 資産売却差額（有価証券の売却益）が **年間4.9億円**
- 教育活動外の運用収益が教育活動を下支えする構造

### 7-3. 継続的な黒字体質
- 5年間すべてで事業活動収支差額がプラス
- 基本金組入前当年度収支差額は毎年度 **4.0〜16.6億円** の黒字（2020年度は資産処分差額9.7億円の特殊要因で低め）
- 繰越収支差額は5年間で **193.4億円 → 243.8億円** に増加（+50.3億円）

## 8. 財務上の課題・リスク

### 8-1. 学生納付金の減少
- 2022年度ピーク27.8億円から2024年度23.5億円へ **15.5%減少**
- 学校関係者評価報告書（2025年10月）において「**18歳人口の減少を受け日本人入学生が減少傾向**」と明記。外国人留学生の増加はあるが総定員に対して減少
- 同報告書の財務評価でも「18歳人口が減少期に移るなど今後の情勢は予断を許さない」「日本人学生の減少は避けることのできない重大（な課題）」と認識
- 教育活動収支差額も2022年度12.0億円 → 2024年度6.9億円へ悪化

### 8-2. 管理経費の増加
- 管理経費は2020年度6.2億円 → 2024年度7.1億円へ増加傾向
- 経常収入に対する管理経費比率20.5%はやや高めの水準

### 8-3. 有価証券への高い依存
- 総資産の34.3%が有価証券（142.3億円）
- 市場変動リスクへのエクスポージャーが大きい

## 10. eduroamインフラ投資への示唆

eduroam導入に必要な投資規模（RADIUSサーバ、AP更新等）は、法人の財務規模に対して微小であり、財務的な障壁は存在しない。

| 項目 | 参考値 |
|------|--------|
| 法人の年間設備関係支出 | 3,629万円（2024年度） |
| 法人の年間施設関係支出 | 3億6,354万円（2024年度） |
| 法人の現金預金残高 | 114.3億円 |
| 法人の年間黒字（基本金組入前） | 15.8億円 |

- 教育研究経費（年間5.2億円）やIT関連投資として十分に吸収可能な規模
- KIC側は既にArubaの統合AP環境を整備済みであり、eduroam対応のSSID追加は追加コスト最小
- 神戸電子側のAP刷新は「教育環境整備」として施設関係支出に計上可能
- 財務的にはeduroam導入の意思決定は **経営判断・組織間調整の問題** であり、予算制約の問題ではない
