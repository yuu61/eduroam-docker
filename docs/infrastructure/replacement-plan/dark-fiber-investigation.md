# NTTダークファイバ・建物間接続

## 背景

神戸電子専門学校（北野地区）のキャンパスでは、本館と北野館・南館間は既存の光ファイバで接続済みだが、共創館・一宮館・マルチメディア館等は物理的に離れている（20m以上、公道を挟む区間あり）。これらの建物間ネットワーク接続にNTTダークファイバの利用可能性を整理する。

---

## 1. NTTダークファイバの制度概要

### 1.1 ダークファイバとは

NTT東西が敷設した光ファイバ設備のうち、未使用（光信号が通っていない）の芯線を、他の電気通信事業者に貸し出す制度。2001年に総務省がNTT東西に対しダークファイバの開放を義務付けたことで制度化された。「ダーク」とは光が通っていない状態を指す。

### 1.2 利用条件 -- 電気通信事業者限定

**最重要ポイント: NTTダークファイバの直接利用は電気通信事業者に限定される。**

| 条件 | 詳細 |
|------|------|
| 対象 | 総務大臣の**登録**または**届出**を受けた電気通信事業者のみ |
| 目的 | エンドユーザへの電気通信役務提供が目的であること |
| 禁止事項 | 事業者自身の**自家利用**目的、**転貸**目的は不可 |
| 学校法人の場合 | 学校法人は電気通信事業者ではないため、**NTTから直接ダークファイバを借りることはできない** |

#### 一般法人がダークファイバを利用する方法

一般法人（学校法人含む）がダークファイバを利用するには、以下のいずれかの方法をとる:

1. **電気通信事業者経由**: ダークファイバをNTTから借り受けた電気通信事業者が、一般法人に「芯線貸し」サービスとして提供する形態。株式会社ネディア等の事業者が大学向けに提供している実績がある
2. **電気通信事業者の届出を行う**: 自ら電気通信事業の届出を行い、電気通信事業者として直接NTTと契約する（ただし第三者への役務提供が前提であり、純粋な自家利用は原則不可）

### 1.3 提供形態

NTT東西のダークファイバは以下の種類がある:

| 提供形態 | 用途 | 概要 |
|---------|------|------|
| **加入光ファイバ（シングルスター方式）** | アクセス区間（局～ユーザビル） | NTT局からユーザビルまで1芯を専有 |
| **加入光ファイバ（シェアドアクセス方式）** | アクセス区間（局～ユーザビル） | 局外スプリッタまで共有、そこからユーザビルまで専有 |
| **中継光ファイバ** | 局間区間（NTT局～NTT局） | NTT局舎間の中継区間を1芯単位で借用 |

光ファイバは**1芯単位**で提供される。

### 1.4 接続料金（2024年4月適用、税抜）

#### NTT西日本（関西エリア）

| 項目 | 月額料金（1芯あたり） |
|------|---------------------|
| 加入光ファイバ（シェアドアクセス方式・主端末回線） | **1,796円** |
| 加入光ファイバ（シングルスター方式） | **2,306円** |
| 加入光ファイバ（シェアドアクセス方式・分岐端末回線） | 468円 |
| 中継光ファイバ | **1.295円/m** |

#### NTT東日本（参考）

| 項目 | 月額料金（1芯あたり） |
|------|---------------------|
| 加入光ファイバ（シェアドアクセス方式） | 約1,746〜1,779円 |
| 加入光ファイバ（シングルスター方式） | 約2,148〜2,162円 |
| 中継光ファイバ（一般光信号中継伝送機能） | 約1.083円/m + 管理費40円/芯 |

**注意**: 上記は電気通信事業者がNTTに支払う「接続料」であり、エンドユーザ（学校法人等）が電気通信事業者経由で利用する場合の料金は、これに事業者のマージンが加算される。

#### 参考計算（キャンパス内利用想定）

北野地区での建物間接続の場合、同一NTT局舎の管轄内で完結する可能性が高い:
- 加入光ファイバ × 2本（A棟側・B棟側、各局～ユーザビル）: 約2,306円 × 2 = 約4,612円/月
- 中継光ファイバ（局間接続が必要な場合）: 距離依存

ただし実際には、接続事業者のサービス料金として上記の数倍になることが一般的。

### 1.5 初期費用

| 項目 | 概算 |
|------|------|
| 光ファイバ引込工事 | 個別見積（数万〜数十万円） |
| 局内接続工事 | 個別見積 |
| 最低利用違約金 | 新規開通日より1年以内の解約で発生 |
| 終端装置（メディアコンバータ等） | 利用者負担（数万〜十数万円） |

### 1.6 申込から開通までの流れと期間

#### 新規事業者の場合のフロー

```
1. 電気通信事業者の登録/届出（総務省）
2. NTTとの接続協議の申し入れ
3. 接続に関する契約の締結
4. 光ファイバ開通申込受付システムにて申込
5. NTTによるルート調査・設計
6. 工事の実施
7. 開通・引渡し
```

#### 標準期間

| フェーズ | 期間 |
|---------|------|
| 接続申込〜開通 | **最大6ヶ月** |
| 納期回答〜開通 | **最大3ヶ月** |
| 開通期日 | 上記いずれか遅い方 |

電気通信事業者経由で芯線貸しサービスを利用する場合は、発注後のキックオフミーティングから**おおよそ4〜7ヶ月程度**が目安。

## 2. NTT管路の借用（管路開放）

### 2.1 制度概要

NTT東西は、所有する電柱・管路・とう道・マンホールへの他事業者の線路敷設に係る設備を開放している。自前の光ファイバケーブルをNTT管路に通すことが可能。

### 2.2 利用条件

| 項目 | 内容 |
|------|------|
| 対象 | 電気通信事業者（ダークファイバと同様の制約） |
| 開放対象設備 | 電柱、管路、とう道、マンホール |
| 工事方式 | NTT委託工事 または 自前工事（事前契約要） |
| 契約形態 | NTT工事: 「共同収容に関する基本契約書」+「個別契約書」<br>自前工事: 「共同収容の自前工事に関する基本契約書」+「個別契約書」 |
| 調査費用 | 管路等利用に関する調査費用は**依頼事業者の実費負担**（契約締結の有無にかかわらず） |

### 2.3 料金体系

管路利用料金は「管路1mあたりの再建設費 x 年経費率 x 占有率」で算定される（具体的な単価はNTT西日本に個別照会が必要）。

電柱共架料金は、1本あたり年額数百円〜数千円程度が一般的（電力会社の事例では1本あたり年額1,000〜2,000円程度）。

### 2.4 問い合わせ窓口

| NTT | 窓口 | 連絡先 |
|-----|------|--------|
| NTT西日本 | 設備利用に関する問い合わせ | **0800-2000116**（9:00〜17:00） |
| NTT西日本 | 添架申請サポートWEB | https://www.ntt-west.co.jp/tenga/ |
| NTT東日本 | 相互接続推進部 接続営業部門 | **03-5359-4111**（9:30〜17:30、土日祝除く） |

### 2.5 北野地区での適用性

**公道横断区間でのNTT管路利用は原則として可能**だが、以下の制約がある:
- 電気通信事業者としての届出/登録が前提
- 既存管路の空き状況（満管の場合は利用不可）
- 調査費用が先行して発生する

## 3. 大学・教育機関でのダークファイバ利用事例

### 3.1 SINET接続でのダークファイバ活用

SINETは国立情報学研究所（NII）が運用する学術情報ネットワークで、各加入機関は最寄りのSINETノードまでのアクセス回線を自己負担で調達する。

| 項目 | 内容 |
|------|------|
| SINET6の規模 | 全国930機関が10Gbps、612機関が1Gbps以下で接続（2024年5月時点） |
| アクセス回線の選択肢 | 専用線、ダークファイバ、商用ISP回線 |
| ダークファイバの優位性 | 専用線サービスと比較して、**速度・コスト両面で優位になるケースがほとんど** |

### 3.2 群馬県内大学の事例（ネディア施工）

| 項目 | 内容 |
|------|------|
| 構成 | 大学拠点 → ダークファイバ → SINETデータセンター（10GBASE-LR） |
| 導入前 | 専用線 1Gbps |
| 導入後 | ダークファイバ経由 **10Gbps** |
| コスト | ランニングコスト**低下**（具体的金額は非公開） |
| 終端装置 | メディアコンバータ |
| 導入期間 | 発注後4〜7ヶ月 |

### 3.3 KIC（神戸情報大学院大学）の状況

- SINET6に**加入済み**
- 接続方式の詳細は要確認（ダークファイバか専用線か）
- KICのSINET接続をテコに、神戸電子側のキャンパス間接続やSINET接続の相乗りも検討可能

## 4. 代替手段の比較

### 4.1 選択肢一覧

| 方式 | 概要 | 月額費用感 | 初期費用感 | 導入期間 | 学校法人が直接契約可能か |
|------|------|-----------|-----------|---------|----------------------|
| **NTTダークファイバ（事業者経由）** | 電気通信事業者がNTTからダークファイバを借り、芯線貸しサービスとして提供 | 数千〜数万円/芯 | 数十万〜百万円 | 4〜7ヶ月 | 間接的に可能 |
| **NTT管路借用** | NTT管路に自前光ファイバを敷設 | 距離依存（低コスト） | 工事費+調査費 | 数ヶ月 | 電気通信事業者のみ |
| **NTT西日本 ビジネスイーサ ワイド** | 広域イーサネットサービス | 1Gbps: 約40万円/月〜 + 中継設備15.5万円/拠点 | 数万円 | 1〜2ヶ月 | **可能** |
| **オプテージ イーサ専用線** | 関西電力系、拠点間専用線 | 100Mbps: 30.6万〜137.4万円/月（距離依存）<br>1Gbps: 108万〜693万円/月 | 数万円 | 1〜2ヶ月 | **可能** |
| **フレッツ・VPN ワイド** | NGN閉域ネットワーク | 月額2,530円〜 + フレッツ光月額 | フレッツ工事費 | 2〜4週間 | **可能** |
| **架空自営光ファイバ** | NTT電柱に自前光ファイバを共架 | 電柱使用料のみ（年額数千円） | 工事費（数十万円） | 数ヶ月 | 電気通信事業者のみ |
| **直接埋設/直接架空** | 道路占用許可を取得し自前で敷設 | 維持費のみ | 道路占用許可+工事費 | 数ヶ月〜 | 要道路管理者許可 |

### 4.2 北野地区の短距離（20〜100m）接続での推奨度

北野地区で公道を挟む建物間接続（短距離）の場合、コストパフォーマンスの観点から:

#### 推奨度: 高

**1. フレッツ光 + フレッツ・VPN ワイド**
- 両方の建物にフレッツ光を引き、フレッツ・VPN ワイドで閉域接続
- 月額: フレッツ光 × 2拠点（各約5,000〜6,000円） + VPN ワイド（2,530円〜）= **約1.3〜1.5万円/月**
- 帯域: ベストエフォート（最大1〜10Gbps）
- 学校法人が直接契約可能
- 導入期間: 2〜4週間
- 欠点: ベストエフォートのため帯域保証なし

**2. NTT西日本 ビジネスイーサ ワイド（同一MA内接続）**
- 同一MA（単位料金区域）内の2拠点をイーサネットで専用接続
- 帯域保証型（1Mbps〜100Gbps）
- 月額: アクセス回線料 × 2拠点 + 中継設備利用料15.5万円（同一MA・1Gbps以下）
  - 例: 100Mbps × 2拠点 = 約22万円/月（概算）
- 学校法人が直接契約可能
- SLA（サービス品質保証）あり

#### 推奨度: 中

**3. ダークファイバ（事業者経由）**
- 帯域を100%専有でき、将来の増速も容易
- ただし20〜100mの短距離接続には「大げさ」
- NTTの加入光ファイバは局〜ユーザビル間であり、**ユーザビル間の直接接続はできない**（両方のビルからNTT局まで光ファイバを引き、局内で接続する構成になる）
- 短距離でもNTT局を経由するため、物理的な距離以上のコストがかかる

#### 推奨度: 低（短距離の場合）

**4. オプテージ イーサ専用線**
- 品質は高いが、短距離でも最低数十万円/月と高コスト
- 大規模キャンパス間（km単位）向き

### 4.3 特記: NTT局を経由しない建物間直接接続

北野地区の20〜100m程度の距離であれば、以下の方法でNTT局を経由せず直接接続できる可能性がある:

1. **NTT電柱への光ファイバ共架**: 公道上のNTT電柱に自前の光ファイバを架設する。ただし電気通信事業者としての届出が必要
2. **電力会社電柱への共架**: 関西電力の電柱に共架する方法。電気通信事業者以外でも「自営電気通信設備」として申請可能な場合がある
3. **道路占用許可による自前敷設**: 道路管理者（神戸市）に道路占用許可を申請し、地下管路を自前で敷設する。学校法人でも申請可能だが、手続きが煩雑
4. **既存の建物間ルート**: 北野地区の建物の物理的な配置によっては、敷地内で完結する引回しルートが存在する可能性もある（要現地調査）

## 6. 結論と推奨アクション

### 6.1 北野地区の建物間接続（20〜100m、公道横断）への適用

| 方式 | 適用性 | 理由 |
|------|--------|------|
| NTTダークファイバ直接利用 | **不可** | 電気通信事業者限定。学校法人は直接利用不可 |
| ダークファイバ（事業者経由） | **オーバースペック** | 短距離接続にはコスト・手続き・期間が見合わない。NTT局経由になるため物理距離以上のコスト |
| NTT管路借用 | **困難** | 電気通信事業者限定 |
| **フレッツ光 + VPN ワイド** | **最有力候補** | 低コスト、短期導入、学校法人が直接契約可能。ベストエフォートだが1〜10Gbps |
| ビジネスイーサ ワイド | **帯域保証が必要なら有力** | SLA付き帯域保証。ただし月額20万円以上 |
| 道路占用許可+自前敷設 | **要検討** | 一度敷設すれば低ランニングコスト。ただし手続きが煩雑で初期コスト大 |

### 6.2 推奨する次のステップ

1. **現地調査**: 対象の建物間に敷地内で完結する引回しルートがないか確認する（公道横断を回避できれば選択肢が大幅に広がる）
2. **NTT西日本 法人営業への問い合わせ**: ビジネスイーサ ワイドとフレッツ・VPN ワイドについて、北野地区の具体的な拠点を示して見積を依頼する
3. **KICのSINET接続構成の確認**: KICがどのようなアクセス回線でSINETに接続しているかを確認し、その回線を共用・拡張できるか検討する
4. **オプテージへの問い合わせ**: eo光ネットの法人プランやイーサ専用線について、短距離接続向けの料金を確認する
5. **SINET加入の検討**: 神戸電子が独自にSINET加入する場合（学校法人コンピュータ総合学園として既にKIC経由で加入資格あり）、ダークファイバによるSINETノード接続と建物間接続を兼ねる構成が最もコスト効率が良い可能性がある
